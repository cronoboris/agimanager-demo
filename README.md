## 게임 소개

AGI Manager는 패러독스(Paradox) 스타일의 AI 스타트업 경영 시뮬레이션입니다.
2017년, 딥러닝이 세상을 바꾸기 시작한 해. 당신은 작은 AI 스타트업의 CEO로서 역사의 한가운데에 서 있습니다.

- 91개 기술 노드의 방대한 기술 트리
- 3D 지구본 위 글로벌 AI 경쟁
- 서비스 가격 전략과 시장 점유율 경쟁
- 데이터센터 건설 (7개 지역 + 우주)
- 14가지 엔딩
